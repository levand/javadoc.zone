Register a new Javadoc host

The Repository URL is the base URL of the Javadoc host you want to register. It may refer either to the index.html file at the root of the Javadoc repository, or to the repository directory itself. For example, both of the following are valid repository URLs:

  • https://google.github.io/guava/releases/19.0/api/docs/
  • https://google.github.io/guava/releases/19.0/api/docs/index.html

The version of the library to which the Javadoc repository refers. For libraries delivered via a public Maven repository, this must be the Maven version string.

The maven artifact coordinates for the library to which this Javadoc repository refers. May be left blank if the relevant library is not delivered through a public Maven repository. Examples:

  • com.google.guava/guava
  • org.apache.commons/commons-math3
