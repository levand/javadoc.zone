This site works by maintaining an index of Java class names to locations on the internet which host relevant Javadocs.

Users can register additional Javadoc hosts, which will then be indexed. Because all Javadoc hosts are ultimately identified by users of this site, I cannot be responsible for inaccurate or malicious results. If you do find one, please file an issue on this site's Github repository and I'll fix it as soon as possible.

If you're interested in the tech behind a simple AWS-hosted serverless webapp, please check out the source code. Feedback and PRs welcome.
